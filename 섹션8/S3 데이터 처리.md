1) S3 이벤트 알림
- S3 버킷에서 특정 이벤트가 발생할 때 알림을 보내는 기능
- 새 객체 생성 이벤트, 객체 제거 이벤트, 복제 이벤트,

- 이벤트 알림을 AWS 서비스로 보낼 수 있음
- S3 이벤트 알림과 수신 대상 서비스를 결합하여 다양한 애플리케이션을 구현 가능

- 이벤트 알림을 허용할 수 있음

- S3 Object lambda
-> 애플리케이션으로 데이터가 반환될 때 데이터를 수정 및 처리하는 기능
-> 자체 코드를 추가하여 S3에서 검색한 데이터를 애플리케이션으로 반환하기 전에 처리할 수 있음
ex) 사용자 지정 코드를 사용하여 이미지 크기 조정, 워터마크 삽입, 파일 형식 변환, 객체 이름 변환 등
-> 애플리케이션을 변경할 필요 없이 S3 Object Lamdba 변환을 사용

- S3 Batch Operation
- Amazon S3 객체에 대해 대규모 배치 작업을 수행하는 기능
- 한 번에 최대 수십억 개의 객체에 대해 단일 작업을 수행하는 기능
- S3 배치 작업에서 지원하는 작업

- 객체 옆에 속성 Tab
-> 이벤트 알림
-> 이벤트 알림 생성

- 이벤트 브리지로도 보낼 수 있음

- 왼쪽 상단의 아이콘을 클릭하면, 객체 Lambda 액세스 지점
-> 액세스 포인트를 만들어서 가능
-> 배치 작업
-> 대규모 배치 작업 수행 가능
-> S3 복제 구성을 사용하여 매니페스트 구성

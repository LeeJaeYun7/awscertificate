1) EC2 라이프 사이클
- EC2 인스턴스의 시작 순간부터 종료될때까지의 다양한 상태
- 시작 - 실행중 - 재부팅 - 중지 - 종료

- 인스턴스 상태
1) 시작: 인스턴스 전원을 켜는 것
2) 재부팅: 인스턴스를 재부팅
3) 중지: 인스턴스 전원을 끄는 것
4) 종료: 인스턴스 삭제
5) 최대절전모드: 노트북에서 종료를 안하고 스크린만 닫는 절전모드와 비슷
                -> RAM에 있는 애플리케이션 상태를 루트볼륨에 저장
                -> 서버 부팅 시 프로세스 로드 시간이 오래 걸리는 시스템의 경우 유용

- 루트 볼륨이 암호화된 EBS 볼륨이어야 함
-> 그래야 최대 절전 모드를 사용할 수 있음
